Create a Web Forms Module

Overview

Note: These instructions use templates to illustrate how to create a web forms module.

Prerequisites

  • A local DNN installation with Host permissions.
  • Visual Studio 2015 is the recommended IDE for developing DNN modules.

Steps

  1. Download and install the templates.
    • If installing Chris Hammond's templates, see his instructions under "Installing the Templates."
    • If installing the DNN 8 templates:
      1. Download the zip file and unzip.
      2. In the templates folder, unzip the file of the specific template you want. Two are included: Dnn Mvc Module and Dnn Spa Module.
      3. Copy the extracted template folder to your DNN installation folder, under Portals > _default > Templates???.
  2. Create a Visual Studio project.
    1. Run Visual Studio as an administrator.
    2. File > New > Project
    3. Choose the following settings:
      1. Expand Installed > Templates > Visual C# or Visual Basic > DotNetNuke.
      2. Choose one of the compiled modules.
      3. Name: the name of your new module
      4. Location: the DesktopModules folder inside your DNN installation folder
      5. Uncheck Create directory for solution and Add to source control.

  3. Modify the Visual Studio project to add functionality to your new module.
