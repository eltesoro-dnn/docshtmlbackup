Contents of a Theme Package

A theme package includes the following:

  • (root folder)
    • Layout template files (*.ascx)
    • Container files (*.ascx)
    • (Optional) CSS for individual layout templates or containers (*.css with the same filename as your layout template or container)
    • (Optional) CSS for the entire theme (skin.css)
    • (Optional) Document Type XML file for individual layout templates or containers (*.doctype.xml)
    • (Optional) Property values of tokens used in your HTML layout template or container (*.xml with the same filename as your HTML layout template or container)

    • Note: You can choose your own folder names for the following.

  • images contains all image files used by the theme, including:
    • (Optional) Preview images of individual layout templates or containers (*.png/jpg with the same filename and in the same folder as your layout template or container)
  • fonts contains all web fonts used by the theme.
  • js contains all JScript files used by the theme, such as bootstrap.min.js and common.js.
  • menu contains all desktop menu files for the theme. If a theme includes multiple menu templates (i.e., a left-hand nav tree and a horizontal menu bar), you can store each menu in its own folder.
