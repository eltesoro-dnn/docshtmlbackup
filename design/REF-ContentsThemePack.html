Contents of a Theme Package

A theme package includes the following:

  • (root folder)
    • Layout template files (*.ascx)
    • Container files (*.ascx)
    • (Optional) Document Type XML file for individual layout templates or containers (*.doctype.xml)
    • (Optional) Custom attributes of tokens (skin.xml)
  • images contains all image files used by the theme, including:
    • (Optional) Preview images of individual layout templates or containers (*.png/jpg/etc. with the same filename as your layout template or container)
  • css contains all CSS required by layout templates and containers, including:
    • (Optional) CSS for individual layout templates or containers (*.css)
    • (Optional) CSS for the entire theme (skin.css)
  • fonts contains all web fonts used by the theme.
  • js contains all JScript files used by the theme, such as bootstrap.min.js and common.js.
  • menu contains all desktop menu files for the theme.
  • mobmenu contains all mobile menu files for the theme.
