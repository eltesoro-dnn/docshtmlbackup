Set Up DNN

Overview

DNN offers three products:

  • DNN Platform is the free open-source package.
  • Evoq Content adds more advanced website management features over the DNN Platform.
  • Evoq Engage adds user engagement features over Evoq Content.

You can compare the features in detail.

Requirements

  • DNN 8.0 Requirements
  • (Optional) Developers: Visual Studio 2015 is the recommended IDE for developing DNN modules.

Steps

  1. Download DNN and set up folder permissions for the user account that will run your website.
  2. Enable and set up IIS.
  3. Install and set up SQL.
  4. Run the DNN Installation Wizard.
