Install an Extension

Overview

A DNN extension can contain a module, a theme, or other components that extend the functionality of the DNN Platform.

You can also create your custom modules and themes.

The extension must be added by the host / super user account to the DNN installation before it can be used in any of the websites.

Prerequisites

Steps

  1. Go to Host > Extensions.
  2. Click Install Extension Wizard.
  3. Click Choose File and select the zip file of the extension. The button name changes to the name of the zip file.
  4. Informational pages will be displayed one after the other:
    1. Package Information includes the extension's version and description, as well as information about the creator.
    2. Release Notes displays what's new in the current release of the extension.
    3. License displays the creator's license terms. Accept the license to continue.
    4. Package Installation Report displays a log of actions performed by the installation.
  5. Verify that the newly installed extension appears in the list under the Installed Extensions tab.
